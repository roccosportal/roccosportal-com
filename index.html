Projects

Pvik

Pvik is just another php5 framework and is more an expirement and learning project than an attempt to create a serious framework in the row with other big frameworks like Symfony, ZendFramework and Fatfree. It follows the MVC concepts and provides a database abstraction layer which supports so far MySQL and MSSQL. Even Pvik is a one person project it has a wiki to help you building websites with it. There is also a pvik boilerplate available. The source code is located on Github. Please feel free to contribute.

Dashbird

Dashbird is an attemp to mix a social network with a pinboard. In earlier stages it was just a pinboard created for my self on which I could left notes for myself. In this phase it already included some features like tagging and searching. More and more of my friends used this service and so it went from single user to multi user. Most of the functionalities are written in Javascript and Pvik is mostly used for saving the data to a MySQL backend. The source code is located on Github.

Pvik-Boilerplate

The boilerplate contains a setup of required folders and files for developing websites with Pvik. You can download it here. The source code is located on Github.

Pvik-Admin-Tools

If you have specifiy a model in Pvik you already have specified which table, columns and relations exists. PvikAdminTools uses this information to build a simple admin backend on top of it. How you set up the tool is described in the wiki. The source code is located on Github.
